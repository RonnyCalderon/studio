# **App Name**: Ignite Desire

## Core Features:

- Weekly Challenge Generation: Generates a random weekly challenge from the list of provided sexy challenges using a shuffle algorithm. This uses a 'tool' to apply logic in selecting the task to make it the most suitable for each partner.
- Challenge Display: Displays the current weekly challenge to both partners with a countdown timer showing the remaining time.
- Challenge Completion Confirmation: Allows either partner to confirm completion of the challenge. Upon confirmation, unlocks a time-limited reward.
- Reward Unlocking: Unlocks a time-limited reward (e.g., access to premium content, special badges, couple-profile customization) upon successful challenge completion.
- Gamified Progression: Tracks challenge completion and rewards earned, providing a visual progression system (e.g., experience points, levels, leaderboards) to gamify the experience.
- Partner Profiles: Allows each partner to create a profile to see their progression. Allow user customizable preference to show what they are comfortable seeing for future preferences for weekly challenges

## Style Guidelines:

- Primary color: Deep crimson (#8B0000) evoking passion and intimacy.
- Background color: Soft beige (#F5F5DC) to create a warm and inviting atmosphere.
- Accent color: Rose gold (#B76E79) for highlights and calls to action.
- Headline font: 'Playfair', a modern serif for an elegant, fashionable feel.
- Body font: 'PT Sans', a humanist sans-serif font with modern feel.
- Use elegant, suggestive icons related to love and desire.
- Use smooth transitions and subtle animations to enhance the feeling of intimacy and excitement.
